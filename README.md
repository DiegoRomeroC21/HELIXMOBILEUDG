# Proyecto de Videojuego en Unity 3D

Este repositorio contiene el proyecto para el desarrollo de un **videojuego en 3D** realizado como parte de la actividad escolar. En este proyecto, se han seguido varias etapas de desarrollo, que se documentarán aquí conforme se vayan completando.

---

# Actividad 4.1: Creación del Proyecto en Unity (Propuesta en 3D)

## Tareas Realizadas:

1. **Visualización de videos**  
   Se consultaron los videos **"Crea tu primer videojuego con Unity"** y **"Crea tu primer videojuego para Android con Unity"** para entender la creación de los recursos necesarios para esta fase del desarrollo.

2. **Creación de la escena principal**  
   Se **creó la escena básica** del juego en 3D, incluyendo los siguientes objetos:
   - **Pelota**: Un objeto que se mueve por la escena e interactúa con las plataformas del helix.
   - **Helix/Cilindro**: El objeto principal sobre el que la pelota se mueve, representando el ambiente del juego.
   - **Pisos**: Se crearon plataformas por las que la pelota va bajando, permitiendo la interacción entre la pelota y el entorno.

3. **Controlador de cámara**  
   Se configuró un sistema para que la **cámara siga a la pelota** mientras se mueve por la escena, asegurando una experiencia visual fluida.

4. **Colisiones**  
   Se **integraron las colisiones** de los pisos y la pelota, permitiendo que la pelota interactúe correctamente con las plataformas y que no atraviese objetos sólidos.

5. **Controlador del Helix**  
   Se creó un **controlador para el helix**, lo que permite que el jugador interactúe con el cilindro (helix) para que la pelota siga un camino específico.
